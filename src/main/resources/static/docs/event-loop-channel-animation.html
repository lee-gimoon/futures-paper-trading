Event log

핵심 구분: TCP 연결은 OS socket으로 유지되고, Netty Channel은 그 socket을 애플리케이션에서 다루기 좋게 감싼 객체다. Step에서 socket과 Channel이 같이 켜져도 같은 연결을 서로 다른 레벨에서 보는 것이고, 실제 Java 실행은 event loop가 켜질 때만 일어난다.
